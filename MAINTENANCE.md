# Maintenance notes

This fork is maintained as a practical AI comic and short-video creation app
experiment. The current focus is to keep the project runnable, document the
setup path, and make the AI workflow easier to test and review.

## Current maintenance scope

- Keep dependency and setup instructions understandable for new users.
- Review API configuration and token-handling paths for safer defaults.
- Triage reproducibility issues in the Flutter app and web build.
- Improve documentation around the script, storyboard, image, and video flow.

## Near-term roadmap

- Add a verified setup checklist for Flutter and web builds.
- Document required API providers and environment variables.
- Add a small smoke test for API configuration parsing.
- Review generated media workflow errors and make failures easier to diagnose.

## Planned Codex usage

Codex would be used for routine open-source maintenance work: issue triage,
small code reviews, release-note drafting, documentation cleanup, and test
generation for the AI workflow. API credits would support prototype testing for
script, storyboard, image, and video pipeline improvements.
